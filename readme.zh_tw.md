# 打包
1. 將此源碼庫克隆到一個空資料夾。
2. 獲取所需版本的gradle並解壓縮到父資料夾。
3. 使用Advanced Installer打開檔案`{version}\{version}-setup.aip`。
4. 在必要時修改版本屬性，並更改檔案同步屬性。
5. 储存為`{version}\{version}-setup.aip`並進行打包。

# 目的
- 用於為HyperCol的開發和生產環境快速部署gradle。
